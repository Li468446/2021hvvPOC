# CVE-2021-21402
### 漏洞描述
Jellyfin 是一个自由的软件媒体系统，用于控制和管理媒体和流媒体。它是 emby 和 plex 的替代品，通过多个应用程序从专用服务器向终端用户设备提供媒体。
Jellyfin 低于 10.7.1 的版本中，攻击者可以通过构造恶意请求从 Jellyfin 服务器的文件系统中读取任意文件。当以 Windows 做为服务器操作系统时，此问题更加普遍。暴露于公共Internet的服务器可能会受到威胁。
解决建议
